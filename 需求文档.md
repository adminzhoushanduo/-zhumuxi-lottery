# 周沐汐的魔法抽奖 — 需求文档

> 版本：v1.0
> 日期：2026-06-14
> 技术栈：HTML+CSS+JS + Node.js(Express) + JSON文件存储 + Railway部署

---

## 一、产品概述

一款专为6岁小朋友（周沐汐）设计的趣味老虎机抽奖游戏。通过每日免费抽奖、完成任务赚取额外机会、
收集钻石和碎片兑换奖励的机制，鼓励小朋友养成良好的日常生活习惯。

---

## 二、核心功能

### 1. 老虎机抽奖 🎰
- 3×3 老虎机面板，外围8个格子 + 中心1个终极大奖
- 每次抽奖播放老虎机滚动动画（格子闪烁 + 哒哒哒音效）
- 抽奖结果弹出卡片展示，终极大奖有彩带庆祝特效

### 2. 奖项系统 🏆
| 格子 | 类型 | 概率 |
|------|------|------|
| 🌟 塞罗奥特曼（中心） | 终极大奖 | 0.1% + 100抽保底 |
| 💎 钻石×20 | 货币 | 15% |
| 💎 钻石×10 | 货币 | 30% |
| 🧩 碎片×2 | 货币 | 5% |
| 🧩 碎片×1 | 货币 | 2% |
| 🍭 棒棒糖 | 小礼物 | 15% |
| 📯 哨子 | 小礼物 | 10% |
| 🙏 没抽到 | 空 | 22.9% |

### 3. 双货币系统 💎🧩
- **钻石：** 收集50个可兑换1次额外抽奖机会
- **碎片：** 收集50个可兑换塞罗奥特曼
- 顶部进度条实时显示收集进度
- 满50时出现兑换按钮

### 4. 任务系统 📋
- 共11个日常生活任务（如收拾玩具、刷牙洗脸、帮忙做家务等）
- 点击"去做任务"→ 弹出文本框填写完成内容 → 提交审核
- 每日自动重置：每天0点，已通过/已拒绝的任务自动变为待完成
- 任务列表每个任务配有🔊语音播报按钮（适合6岁不识字的小朋友）

### 5. 管理员审核 🔐
- 独立管理后台：/admin.html（密码：5201314）
- 显示待审核任务及小朋友填写的完成内容
- 审核通过 → 额外抽奖次数+1
- 审核拒绝 → 任务可重新完成
- 支持手动刷新任务按钮

### 6. 语音播报 🔊
- 任务喇叭按钮：点击后用浏览器语音合成读出任务名称和描述
- 抽奖音效：老虎机滚动哒哒哒、中奖叮叮叮、终极大奖华丽旋律
- 说话时喇叭按钮有闪烁动画

### 7. 数据存储 💾
- 前端：localStorage 作为本地缓存
- 后端：Node.js + Express + data.json 文件存储
- 多设备数据同步（手机/电脑共享同一份数据）
- API 接口：GET/POST /api/data

## 五、数据模型

```json
{
  "lastDrawDate": "2026-06-14",     // 上次免费抽奖日期
  "extraChances": 0,                // 额外抽奖次数
  "diamond": 20,                    // 钻石数量
  "shardCount": 4,                  // 碎片数量
  "pityCount": 18,                  // 保底计数
  "ownedGifts": [],                 // 已获得礼物
  "lastResetDate": "2026-06-14",    // 上次任务重置日期
  "firstVisit": false,              // 是否首次访问
  "history": [                      // 抽奖历史
    {"type":"dia","amount":10,"date":"2026-06-12","name":"钻石×10","emoji":"💎"}
  ],
  "tasks": [                        // 任务状态
    {"id":"t1","status":"todo","content":"","submitDate":null},
    {"id":"t2","status":"approved","content":"...","submitDate":"2026-06-12"}
  ]
}
```
> 任务状态流转：todo → (填写内容提交) → submitted → approved/rejected

## 六、规则说明

1. **每日免费：** 每天1次免费抽奖，跨日自动重置
2. **额外次数：** 完成任务提交审核通过 +1，50钻石兑换 +1
3. **终极大奖保底：** 连续99次没中，第100次必中塞罗奥特曼
4. **任务每日重置：** 每天自动重置已通过/已拒绝的任务
5. **兑换规则：** 50钻石 → 1次抽奖 / 50碎片 → 1个塞罗奥特曼
